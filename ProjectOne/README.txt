TUITION PROJECT 

Using front end web site to make HTTP requests to Python REST API and receive responses for dynamic
	web pages. Python includes the package Flask for receiving and send HTTP. The Python code connects
	with AWS PostgreSQL database. The web pages will include HTML,CSS, JavaScript, and AJAX to 
	send/receive to the Python API. The API will manage PUT, GET, POST, DELETE HTTP requests. The Python 
	code includes Python unit testing. Cucumber testing will automated web service requests by Selenuim, 
	for the expected response data.
